Portfolio - Salini A P

This repository contains the code for a personal portfolio website for Salini A P, designed and developed using pure HTML, CSS, and JavaScript. It showcases Salini's education, projects, certifications, and contact information in a simple and elegant design.

Features

Responsive Design: The website adapts to various screen sizes, making it accessible on desktops, tablets, and smartphones.
Navigation Bar: A fixed navigation bar that allows users to smoothly navigate through different sections of the page.
Home Section: Displays the name of the portfolio owner with a minimalistic header.
About Me Section: Includes a brief description of Salini's background, technical skills, and professional experience with a profile photo.
Education Section: Lists educational qualifications with time periods and details.
Projects Section: Highlights notable projects with descriptions and key features.
Certifications Section: Lists certifications with clickable links to view the details of each certification.
Contact Section: Contains contact information and a simple contact form for communication.

Technologies Used

HTML: Structure of the website.
CSS: Styling and layout of the website.
JavaScript: Interactive functionality for form submission and other dynamic elements.
